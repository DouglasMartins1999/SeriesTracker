# SeriesTracker

O SeriesTracker é um aplicativo POC - Prova de Conceito - desenvolvido nativamente para dispositivos iOS. A aplicação disponibiliza ao usuário uma lista das séries adicionadas ao aplicativo, separando-as entre as concluídas e as em andamento. Para cadastrar uma série, o usuário apenas precisa informar o nome, quantidade total de episódios e quantidade de episódios assistidos.

A aplicação foi desenvolvida como um dos primeiros contatos com a linguagem Swift, ao longo de uma aula do curso presencial “Foundation”, disponibilizado pela [Apple Developer Academy](https://developeracademy.mackenzie.br/) aos alunos da Universidade Presbiteriana Mackenzie. A aplicação teve o iPhone 11 como alvo, no entanto, foi adaptada posteriormente para outros modelos de iPhone. É capaz de funcionar em dispositivos com iOS 12.4.


## Como posso testar?

Dispositivos iOS não permitem instalar aplicativos gratuitamente fora da App Store, portanto, não é possível instalar esse aplicativo caso você não seja um desenvolvedor. Desenvolvedores precisam de um computador rodando MacOS e do XCode instalado para editar e executar a aplicação em um simulador ou em seu próprio aparelho com iOS. Se for seu caso, apenas clone esse repositório e abra o projeto “SerieList.xcodeproj”.
